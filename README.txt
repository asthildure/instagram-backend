# Firebase backend for Instagram Link Saver

## Slik setter du opp:

1. Gå til https://console.firebase.google.com
2. Opprett prosjekt og aktiver Firestore (test mode)
3. Gå til Project Settings > Service Accounts
4. Generer og last ned serviceAccountKey.json
5. Last opp denne filen til backend-prosjektet
6. Deploy til Render som før

⚠️ Ikke legg serviceAccountKey.json på GitHub offentlig!
